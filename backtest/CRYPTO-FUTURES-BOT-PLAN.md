### ⭐ СТРАТЕГИЯ #1 — Funding Rate Arbitrage (Нейтральная)

**Тип:** Delta-Neutral / Арбитраж  
**Сложность:** Средняя  
**Риск:** Низкий  
**Ожидаемая доходность:** 15–60% годовых

**Принцип работы:**
```
Perpetual futures имеют механизм Funding Rate.
Если рынок в лонге → ставка положительная → ШОРТИСТЫ получают выплаты.

Стратегия:
  SPOT BUY (например, 1 BTC) + FUTURES SHORT (1 BTC)
  = Delta Neutral (цена не важна)
  = Получаешь Funding Rate каждые 8 часов
```

**Формула доходности:**
```
APR = Funding Rate (%) × 3 выплаты/день × 365 дней
При ставке 0.01%/8h → APR = 0.01% × 3 × 365 = 10.95%
При ставке 0.05%/8h → APR = 0.05% × 3 × 365 = 54.75%
```

**Исторические данные (Binance BTC Perpetual):**
- 2021 bull run: avg funding 0.03–0.08% per 8h → 40–100% APR
- 2022 bear: иногда отрицательный (лонги платят шортам)
- 2023-2024: средний 0.01–0.03% → 10–30% APR

**Риски:**
- Liquidation если не хватает маржи на спот/фьючерс позиции
- Funding может стать отрицательным (нужен стоп)
- Exchange risk (держишь крипту на бирже)

**Когда работает:** Бычий рынок, высокая волатильность, ажиотаж.  
**Когда НЕ работает:** Медвежий рынок с отрицательным funding.

### ⭐ СТРАТЕГИЯ #2 — Trend Following Momentum (Основная)

**Тип:** Трендовая  
**Сложность:** Средняя  
**Риск:** Средний  
**Ожидаемая доходность:** 30–150% годовых (с плечом)

**Экспертный консенсус:** Самая изученная и доказанная стратегия. AQR Research, Winton, Man AHL.  
В крипто работает лучше чем в акциях из-за неэффективности рынка.

**Параметры (оптимальные по исследованиям):**
```python
# Топ конфигурация по backtests (Crypto 2020-2024)
EMA_FAST  = 21      # Быстрая EMA
EMA_SLOW  = 55      # Медленная EMA
ATR_PERIOD = 14     # ATR для стопов
ATR_MULT  = 2.5     # Стоп = ATR × 2.5
ADX_FILTER = 25     # Входить только если ADX > 25 (сильный тренд)
RSI_FILTER = (40, 60)  # Избегать перекупленность/перепроданность

# Управление позицией
RISK_PER_TRADE = 0.01   # 1% капитала на сделку
MAX_LEVERAGE = 3        # Не более 3x (даже если биржа даёт 125x!)
MAX_POSITIONS = 5       # Одновременно не более 5 позиций
```

**Лучшие пары для стратегии (по волатильности и ликвидности):**

### ⭐ СТРАТЕГИЯ #3 — Statistical Arbitrage (BTC/ETH Spread)

**Тип:** Market Neutral / Пары  
**Сложность:** Высокая  
**Риск:** Низкий-Средний  
**Ожидаемая доходность:** 20–50% годовых

**Принцип:**
```
BTC и ETH высоко коррелированы (r = 0.85–0.95).
Когда спред расширяется → сделка на сближение.

Лонг ETH/USDT Perp + Шорт BTC/USDT Perp
(или наоборот, в зависимости от направления отклонения)

Индикатор: Z-score спреда log(ETH/BTC)
Вход:    Z-score > +2.0 или < -2.0
Выход:   Z-score → 0
Стоп:    Z-score > ±3.0 (спред продолжает расширяться)
```

**Доходность:** Более стабильная, меньше drawdown. Идеально для медвежьего рынка.

### ⭐ СТРАТЕГИЯ #4 — Grid Trading (Диапазонные рынки)

**Тип:** Автоматический маркетмейкинг  
**Сложность:** Низкая  
**Риск:** Низкий (при правильных диапазонах)  
**Ожидаемая доходность:** 10–40% годовых

**Принцип:**
```
Устанавливаешь сетку ордеров вокруг текущей цены.

Цена 50,000 BTC. Сетка каждые 1%:
BUY  51,500 → SELL 52,015
BUY  50,500 → SELL 51,005
BUY  49,500 → SELL 50,005    ← текущий диапазон
BUY  48,515 → SELL 49,005
BUY  47,549 → SELL 48,025

Каждое колебание = прибыль от спреда.
```

**Работает:** В боковике (50–70% времени в крипто)  
**Не работает:** При сильном тренде вниз (все лонги становятся убытком)

**Сложность:** Очень высокая

**Ожидаемая доходность:** 50–200% (но высокая дисперсия)
